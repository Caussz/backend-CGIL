## Clone esse projeto

```bash
> git clone https://github.com/Caussz/backend-CGIL
```

## Instale as dependencias do site:

```bash
> cd backend-CGIL
> npm install or npm i
```

### Para rodar
```bash
> npm start
```
### Para Usar PM2
```bash
> npm i -g pm2 (para instalar)

> Para executar = pm2 start vite
> depois = pm2 monit
```
